# Maize-health

This repo is a part of my final year project, it is a pesticide recomendation system that uses AI to scan a maize leaf for pest, detect the pest and recommend a suitable pesticide, it was built using react, vite, firebase and a roboflow pest detection model 

## 🚀 Features

- responsive UI
- loader
- error handling
- image upload
- pest detection
- pesticide recommendation
- integration with firebase
- integration with roboflows pest detecetion model


## 📋 Prerequisites

- Node.js >= 18
- npm or yarn


## 🛠️ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/tobiayinmiro23/Maize-health.git
   ```

2. **Enter the directory**
   ```bash
   cd Maize-health
   ```
   
3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```
